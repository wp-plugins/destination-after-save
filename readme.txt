Allows user to choose destination after saving the page they are editing, or to leave the page they are editing for a new destination.

== Description ==

Aren’t you tired of saving a page, going back to the list of pages, and selecting the next page you need to edit?

Destination After Save cuts down on unnecessary page loads and saves you time.

Don’t need to save? Select the next page you’d like to edit and click the Leave button to depart for your next destination.

== Installation ==

1. Upload the `/destination-after-save/` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
