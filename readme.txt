Allows user to choose destination after saving the page they are editing, or to leave the page they are editing for a new destination.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `/destination-after-save/` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Does this support posts also? =

Not at this time, but future versions will support posts and custom post types.
